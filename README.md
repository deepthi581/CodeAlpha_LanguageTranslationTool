# Language Translation Tool

## Overview

This project is an AI-powered Language Translation Tool developed using Python, Streamlit, and Deep Translator. It allows users to translate text between multiple languages in real time through a simple and user-friendly interface.

## Features

* Translate text between multiple languages
* Real-time translation
* Character count display
* User-friendly interface

## Technologies Used

* Python
* Streamlit
* Deep Translator

## Installation

```bash
pip install streamlit deep-translator
```

## Run the Project

```bash
streamlit run app.py
```
